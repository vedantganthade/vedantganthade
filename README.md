## Selected Projects

### ElderSafe Vision &nbsp; `Funded` `Patent Filed`
Real-time, sensor-free fall detection for elderly monitoring. Camera feed processed through human detection → pose estimation → fall classification → emergency alert. No wearables required.

`Python` `TensorFlow` `OpenCV` `Pose Estimation`

---

### Breast Cancer Segmentation
End-to-end medical image segmentation pipeline. CLAHE preprocessing for contrast enhancement, VGG16 for feature extraction, U-Net architecture for pixel-level segmentation of mammography scans.

`TensorFlow` `U-Net` `VGG16` `CLAHE` `OpenCV`

---

### Brain Tumor MRI Classification &nbsp; `96% Accuracy`
Multi-class CNN classifier for MRI brain scans. Trained on public datasets; published at IEEE IC3I 2022.

`TensorFlow` `CNN` `Medical Imaging` `OpenCV`

---

### Student Attentiveness Monitor
Classroom attention analysis using real-time facial landmark tracking and eye-state detection. Multithreaded pipeline designed for low-latency inference on standard hardware.

`Python` `OpenCV` `Dlib` `Multithreading`

---

### Student Dropout Prediction
Ensemble learning model to identify at-risk students from academic and behavioral signals. Exposed via REST API with a MySQL backend for institutional integration.

`Scikit-Learn` `FastAPI` `MySQL` `Ensemble Methods`
